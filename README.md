# Pre-run Prompt
A simple python utility to show a dialog before running command.

## Flags
### -s `<size>`
***"fullscreen"*** for full screen or ***"#x#"*** for a given size.

### -m `<msg>`
Message to show

### -c `<cmd>`
Command to run

### -t '<seconds>'
Seconds before the prompt goes away and runs command
